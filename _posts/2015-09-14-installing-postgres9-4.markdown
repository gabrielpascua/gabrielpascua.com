<pre>

    # Download the 9.4 Linux x86-x64 installer at
    # http://www.enterprisedb.com/products-services-training/pgdownload

    $ cd ~/Downloads
    $ chmod 740 [DOWNLOADED RUN FILE].run
    $ sudo ./[DOWNLOADED RUN FILE].run

    # Go through the installation
