Installing Postgres

I thought apt-get install postgresql was enough. What I didn't know was that the command installs the Postgres 9.3 version. I can't use that because our application uses the jsonb data type and it wasn't until 9.4 when it was introduced.

  1. Download the 9.4 Linux x86-x64 installer.
  2. $ cd ~/Downloads
  3. $ chmod 740 [DOWNLOADED RUN FILE].run
  4. $ sudo ./[DOWNLOADED RUN FILE].run
  5. Go through the installation

References:

  • http://www.postgresql.org/download/linux/ubuntu/
