LAMP Stack

We have MAMP Pro at work and if you have used it, you know how easy it is to set up local development environments. XAMP is a no-go since it doesn't have a feature to support different PHP versions out of the box. The safe choice for me if I had to install different PHP versions was to use what is native to the OS. LAMP it is.

  1. $ sudo apt-get update
  2. $ sudo apt-get install lamp-server^
  3. Enable localhost:
    $ sudo gedit /etc/apache2/conf-available/fqdn.conf
    > ServerName localhost
    $ sudo a2enconf fqdn
  4. Change the localhost's folder location:
    1. $ sudo cp /etc/apache2/sites-available/000-default.conf /etc/apache2/sites-available/local.conf
    2. $ sudo gedit /etc/apache2/sites-available/local.conf
    3. Change the folder location path to where you want it to be
    4. $ sudo a2dissite 000-default && sudo a2ensite local
    5. $ service apache2 restart
