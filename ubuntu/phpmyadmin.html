Installing PhpMyAdmin

Since I already have a running MySQL, all I needed to do was pick out a client. I was really hoping that Sequel Pro would be available for Ubuntu. I was looking at wikipedia for SQL clients that have support for at least MySQL and Postgres and have fairly recent updates. SQuirreL SQL met my requirements but I don't like the GUI - it looks dated and unorthodox.

  1. $ sudo -i
  2. $ apt-get phpmyadmin
  3. $ gedit /etc/apache2/apache2.conf
  4. Add this at the very last line then save your changes:
    > Include /etc/phpmyadmin/apache.conf
  5. Enable the php dependencies
    1. $ gedit /opt/phpfarm/inst/php-[MAJOR-MINOR-PATCH]]/lib/php.ini
    2. Enable these extensions:
      > extension="/usr/lib/php5/20121212/mysqli.so"
      > extension="/usr/lib/php5/20121212/mcrypt.so"
  6. $ service apache2 restart
