Increasing Grub's Font Size

Related to my 4K display, this one was annoying even if I just had to press the arrow and enter keys to select an OS. Some recommend that you change the ttyl resolution but that sounds too involved for something minor. I am way behind my schedule so I chose to sacrifice this.

  1. $ sudo grub-mkfont --output=/boot/grub/grubfont.pf2 --size=36 /usr/share/fonts/truetype/dejavu/DejaVuSansMono.ttf
  2. $ sudo gedit /etc/default/grub
  3. Copy and paste this line:
    > GRUB_FONT=/boot/grub/grubfont.pf2
  4. $ sudo update-grub
  5. Restart
